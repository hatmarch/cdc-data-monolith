Demo Walkthrough: Strimzi and AMQStreams Operator

Strimzi and AMQStreams Operator

  1. Go to the cdc-sandbox project, Developer Perspective

  2. Click on +Add select Developer Catalog and then start searching for kafka in the "Operator Backed" items

  3. A page will show up, just click Create

  4. Show some of the options (perhaps even "YAML View") briefly, but create a kafka using the defaults, setting name to example then scrolling down and clicking Create

  5. The Kafka cluster will start to appear in the Topology View

  6. Next use the search functionality to search for the Kafka custom resource you just created

  7. Click on example and scroll to the bottom to review the Conditions. Wait until this is set to Ready

  8. Finally, navigate back to the topology view to see all the aspects that were created. Point out

    • Example Zookeeper

    • Example Kakfa (Brokers, see Services)
